Element

Ett element är ett gemensammt namn för alla taggar inom HTML. Det defineras av en starttag, någonting inom taggen och en sluttag.

HTML

HyperText Markup Language eller HTML förkortat, är grundläggande för stor del av internet. Det är ett märkspråk vilket innebär att det bara ger struktur och kan inte utföra kommandon.

ID

ID kan användas för att ge olika attributer till samma typ av element. Ett exempel på detta är att ge h1 och h2 till header-tagar.

Code snippets

Code snippets är en bit av kod som kan återanvändas.

Class

Classer kan ges till element för att ge attributer till elementet. Exempel på detta är att ge en div class="container".

Meta

Meta används för att beskriva metadata inom ett HTML dokument.

Responsive

Att ha en "Responsive" hemsida innebär att den automatiskt adapterar sig till olika typer av enheter.

http

HyperText Transfer Protocol eller http överför data mellan sidor och mottagaren.

https

Https står för Hypertext Transfer Protocol Secure och är en mer säker verision av http.

Compressed data

Data kompression används för att minska antalet bitar för att presentera datan, vilket sparar på utrymme och gör överförningen av datan snabbare.

CDN

Content Delivery Network eller CDN minskar avståndet från en server och användar. Det gör så att en hemsida kan ladda mycket snabbare runt om i världen.

Local bootstrap

Local bootstrap är bootstrap du själv har laddat ner på din dator. Motsatsen kan ses i t.ex. programmerings hemsidor där bootstrap redan finns.

JQuery

Jquery har funktionen av ett javascript biblotek. Den fungerar genom att omvandla längre och komplicerad kod till något kortare.

Flexbox

Flexbox är ett system som kan förenkla fritt flyttande av lådor och containrar utan att behöva använda komplicerad kod.

Grid

Grid är likt flexbox ett system för design och i detta fall tar det formen av ett rutnät.
